Tap Out is a TAP-Y/J report harness written in Ruby.
Read the Documentation

TAP-Y/J is a YAML/JSON streaming document specification for test results.
Read the Specification

Tap Out let's turn drab black and white test output into glorious techno-color.

Select from a wide variety of output formats from the traditional progress reporters to sophisticated interactive debugging reporters.

Tap Out follows the Unix Way consuming TAP-Y/J streams via a shell pipe.
